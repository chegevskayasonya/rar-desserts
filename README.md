# RAR — Авторська кондитерська

Лендінг для бренду корпусних десертів ручної роботи з Києва.  
Повністю кастомний дизайн без шаблонів — від концепції до деплою.

🔗 [Live Demo](https://rar-desserts-kf5b-8fxrisx4t-chegevskayasonyas-projects.vercel.app/)

## Що було зроблено

Клієнт потребував сайт, який продає сам по собі — без менеджера, без зайвих кліків.  
Завдання: показати продукт красиво, розповісти історію бренду та зібрати замовлення прямо на сайті.

Результат — конверсійний лендінг з унікальним візуальним стилем, анімаціями та вбудованою формою замовлення.

---

## Особливості проекту

- Кастомний курсор з lag-ефектом та реакцією на hover
- Плавні scroll-анімації через IntersectionObserver
- Інтерактивна форма замовлення — вибір позицій, підрахунок суми, відправка
- Marquee-банер, лічильник статистики з анімацією цифр
- Темна та світла тема — зберігається між сесіями
- Повна мобільна адаптація, hamburger-меню
- Single-file архітектура — весь проект в одному HTML-файлі
